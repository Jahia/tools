`tools-test-with-definitions-cnd.jar` is generated by building the `./tools-test-with-definitions-cnd/` project:
```
cd tools-test-with-definitions-cnd
mvn package
cp target/*.jar ../tools-test-with-definitions-cnd.jar
```
